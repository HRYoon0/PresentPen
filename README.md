# MacZoomIt 사용설명서

**MacZoomIt**은 프레젠테이션과 화면 공유를 위한 macOS용 화면 주석 도구입니다.
Windows의 ZoomIt에서 영감을 받아 제작되었습니다.

## 시작하기

### 설치
1. `MacZoomIt.app`을 응용 프로그램 폴더로 이동
2. 앱 실행 시 **접근성 권한** 요청 → 시스템 설정에서 허용 필요

### 기본 사용법
- 앱 실행 후 메뉴바에 아이콘이 표시됩니다
- 단축키로 각 기능을 활성화/비활성화합니다
- **ESC** 키로 모든 모드를 종료합니다

## 그리기 모드

화면 위에 자유롭게 그림을 그릴 수 있습니다.

### 활성화
| 단축키 | 설명 |
|--------|------|
| `Ctrl + 1` | 그리기 모드 켜기/끄기 |

### 그리기 도구
| 키 | 도구 |
|----|------|
| `P` | 펜 (자유 곡선) |
| `L` | 직선 |
| `A` | 화살표 |
| `R` | 사각형 |
| `E` | 원/타원 |

### 색상 변경
| 키 | 색상 |
|----|------|
| `1` | 빨강 |
| `2` | 주황 |
| `3` | 노랑 |
| `4` | 초록 |
| `5` | 파랑 |
| `6` | 보라 |
| `7` | 흰색 |
| `8` | 검정 |

### 기타 조작
| 단축키 | 설명 |
|--------|------|
| `Cmd + Z` | 실행 취소 |
| `Ctrl + Shift + C` | 전체 지우기 |
| 스크롤 | 선 굵기 조절 |

## 줌 모드

화면을 확대하여 세부 내용을 보여줄 수 있습니다.

### 활성화
| 단축키 | 설명 |
|--------|------|
| `Ctrl + 2` | 줌 모드 켜기/끄기 |

### 조작법
| 조작 | 설명 |
|------|------|
| 스크롤 위/아래 | 확대/축소 |
| 마우스 드래그 | 화면 이동 (패닝) |

### 줌 모드 내 추가 기능
줌 모드가 활성화된 상태에서 다른 기능을 함께 사용할 수 있습니다:

| 단축키 | 설명 |
|--------|------|
| `Ctrl + 1` | 줌 화면 위에 그리기 |
| `Ctrl + 3` | 줌 화면에서 커서 하이라이트 |
| `Ctrl + 4` | 줌 화면에서 스포트라이트 |

## 커서 하이라이트

마우스 커서 주변을 강조 표시하여 시청자가 커서 위치를 쉽게 파악할 수 있습니다.

### 활성화
| 단축키 | 설명 |
|--------|------|
| `Ctrl + 3` | 커서 하이라이트 켜기/끄기 |

### 스타일 변경
| 단축키 | 설명 |
|--------|------|
| `Ctrl + Shift + 3` | 스타일 순환 (링 → 헤일로 → 채움 → 스퀴클) |
| `Ctrl + Option + 3` | 색상 변경 |

### 크기 조절
| 조작 | 설명 |
|------|------|
| `Ctrl + 스크롤` | 하이라이트 크기 조절 |

### 스타일 종류
- **링**: 테두리만 있는 원
- **헤일로**: 그라데이션 후광 효과 (기본)
- **채움**: 반투명하게 채워진 원
- **스퀴클**: 둥근 사각형

---

## 스포트라이트

커서 주변만 밝게 표시하고 나머지는 어둡게 처리하여 특정 영역에 집중할 수 있습니다.

### 활성화
| 단축키 | 설명 |
|--------|------|
| `Ctrl + 4` | 스포트라이트 켜기/끄기 |

### 추가 기능
| 단축키 | 설명 |
|--------|------|
| `Ctrl + Shift + 4` | 스포트라이트 줌 토글 (돋보기 효과) |

### 특징
- 스포트라이트 활성화 시 마우스 속도가 자동으로 느려져 정밀한 조작이 가능합니다
- 스크롤로 스포트라이트 크기를 조절할 수 있습니다

## 타이머

전체 화면 카운트다운 타이머로 발표 시간을 관리할 수 있습니다.

### 활성화
| 단축키 | 설명 |
|--------|------|
| `Ctrl + 5` | 타이머 모드 시작 |

### 시간 설정 (시작 전)
| 조작 | 설명 |
|------|------|
| `1` | 1분 |
| `3` | 3분 |
| `5` | 5분 |
| `0` | 10분 |
| `↑` / `↓` | 1분 단위 증가/감소 |
| 스크롤 | 세밀한 시간 조절 |

### 타이머 조작
| 키 | 설명 |
|----|------|
| `Space` 또는 `Enter` | 타이머 시작 / 일시정지 / 재개 |
| `ESC` | 타이머 종료 |

### 특징
- 남은 시간이 1분 미만이면 숫자가 빨간색으로 변경됩니다
- 시간이 종료되면 화면이 깜빡이며 알림음이 재생됩니다

## 시스템 요구사항

- macOS 13.0 (Ventura) 이상
- 접근성 권한 필요

---

## 문제 해결

### 단축키가 작동하지 않아요
1. 시스템 설정 → 개인 정보 보호 및 보안 → 접근성
2. MacZoomIt 앱에 권한 부여
3. 앱 재시작

### 마우스 속도 조절이 안 돼요
접근성 권한이 제대로 부여되었는지 확인하세요.
